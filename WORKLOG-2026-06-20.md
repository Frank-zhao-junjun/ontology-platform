### API 端点

| Method | Path | 说明 |
|--------|------|------|
| `POST` | `/api/v1/agents/tasks` | 提交 Agent 任务 |
| `GET`  | `/api/v1/agents` | 列出可用 Agent |

### 请求示例

```json
POST /api/v1/agents/tasks
{
  "agentType": "kimi",
  "prompt": "分析项目结构",
  "cwd": "D:\\AI\\ontology-platform",
  "timeout": 300
}
```
